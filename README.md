This is my [personal website](https://portfolio-website-one-virid.vercel.app/). 
Built using [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

Clone the project first, then follow these steps.

**Setup project and install all packages**
`npm install`

**Compile and build project**
`npm run build`

**Now run the development server:**
`npm run dev`

Open [http://localhost:3000](http://localhost:3000) or whichever port was assigned.
